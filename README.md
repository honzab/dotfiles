# Jan Brucek's dotfiles

## Installation

```
xcode-select --install
brew install cmake vim pyenv-virtualenvwrapper
pip install virtualenvwrapper
git clone https://github.com/honzab/dotfiles.git ~/.dotfiles
cd ~/.dotfiles && script/bootstrap
```
